## Setup

1. Clone the repository

```bash
git clone https://github.com/mirumee/mirumee-workshop.git
```

2. Install the packages

```bash
npm i
```

3. Run

```bash
npm run dev
```

4. Open your browser at `http://localhost:3000/`
